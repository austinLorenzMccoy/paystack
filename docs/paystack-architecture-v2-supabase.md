### Row Level Security (RLS)

All database access is secured at the **row level**:

```sql
-- Users can only see their own data
CREATE POLICY "Users see own content"
  ON content
  FOR SELECT
  USING (auth.uid() = creator_id);

-- Creators see payments for their content
CREATE POLICY "Creators see own payments"
  ON payments
  FOR SELECT
  USING (
    content_id IN (
      SELECT id FROM content WHERE creator_id = auth.uid()
    )
  );
```

**Benefits:**
- âœ… Security enforced at database level
- âœ… No way to bypass (unlike app-level auth)
- âœ… Automatic for all queries
- âœ… Multi-tenant by default
